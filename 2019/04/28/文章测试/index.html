baseUrl

全局错误信息

错误码 说明
200 成功
400 认证失败
401 参数错误
500 系统错误

Rsponse返回体

1.创建文档

请求路径

GET /shiro-foxit/createDocument

头部信息

content-type : application/x-www-form-urlencoded

参数列表

参数名 必填 类型 描述
filename true String 文件名
pageCount true int 文档页数
previewStart true int 文档预览起始页
previewEnd true int 文档预览结束页
ticket true String 鉴权票据

返回data信息

属性 类型 描述
docId String 文档ID
docPassword String 文档密码（偏移14位）
permissionId int 10001 只读 目前只有这个权限
permissionDesc String 权限文本描述
productId int 产品对应的ID

错误码

错误码 说明
402 未实名认证

2.更新文档状态

请求路径

GET /shiro-foxit/updateDocumentStatus

头部信息

content-type : application/x-www-form-urlencoded

参数列表

参数名 必填 类型 描述
status true int 文档状态 2：成功 3：失败
docId true String 文档ID
ticket true String 鉴权票据

错误码

错误码 说明
501 文档更新失败

3.获取文档信息

请求路径

GET /getDocumentInfos

头部信息

content-type : application/x-www-form-urlencoded

参数列表

参数名 必填 类型 描述
docId true String 文档ID

返回data信息

属性 类型 描述
docId String 文档ID
docPassword String 文档密码（偏移14位）
pageCount int 文档总页数
previewStart int 预览起始页
previewEnd int 预览结束页
createTime String 文档创建时间
isLockSys Boolean 是否被管理员锁定
isLockAuthor Boolean 是否被作者锁定

错误码

错误码 说明
502 未找到该文档！

4.获取读者权限

请求路径

GET /shiro-foxit/getReaderPermission

头部信息

content-type : application/x-www-form-urlencoded

参数列表

参数名 必填 类型 描述
docId true String 文档ID
ticket true String 鉴权票据

返回data信息

属性 类型 描述
permissionId int 读者权限 （10001 只读 目前只有这个权限）

错误码

错误码 说明
503 读者无权限！

5.事件统计接口

请求路径

POST http://dcs.foxitreader.cn/pdf2word/actionid

头部信息

content-type : application/json

请求体

格式：client_time$action_id$value$uuid$mac$version$channel$product_name$user_id

//取不到的数据在对应位置填空。$符号不能少；

参数 值 说明
action_id 100001 启动量
100002 安装量
100003 卸载量
300006 保护成功
300007 保护失败
300008 认证成功
300009 认证失败
300010 壳文件打开
300011 壳文件点击下载
300012 保护文档打开
300013 保护文档点击登录
300014 保护文档点击阅读
300015 保护文档点击关闭付费弹窗
product_name ProtectMstr
